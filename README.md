## 桌面便签(墨水屏2.13寸)

选中ESP32-C3芯片，基于ESP32-IDF框架开发  
- 尺寸： 2.13 inch  
- 外形尺寸（裸屏）：59.2mm × 29.2mm × 1.05mm  
- 外形尺寸（驱动板）：65mmx30.2mm  
- 显示尺寸：48.55mm × 23.71mm  
- 工作电压：3.3V/5V 
- 通信接口：SPI  
- 点距：0.194* 0.194  
- 分辨率：250*122  
- 显示颜色：黑、白  
- 局部刷新 ：0.3s  
- 全局刷新 ：2s  

### 连接说明：
| e-Paper | ESP32-C3 | 说明                             |
| ------- | ------------- | -------------------------- |
| Vcc     | 3.3V      | 供电引脚                        |
| GND     | GND       | 地线                            |
| DIN     | 7         | MOSI ESP32-C3主机输出            |
| CLK     | 6         | SPI 时钟                        |
| CS      | 10        | 当CS为低电平的时候，芯片使能        |
| DC      | 9         | 当DC=0时写入命令；DC=1时写入数据    |
| RST     | 18        |复位                             |
| BUSY    | 8         | input=1 忙碌                    |
